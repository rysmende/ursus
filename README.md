# ursus.kg

Single-page website for **URSUS** — a dairy factory in Belovodsk, Chui Region, Kyrgyzstan,
producing cheese, milk, butter and cream.

Hosted on Cloudflare Pages. Every push to `main` triggers an automatic deploy.

## Features
- Trilingual interface — **Russian / Kyrgyz / English** (in-page switcher, no reload)
- Modern dairy-themed design with a product catalog (filterable by category)
- Scroll-driven 3D cheese wheel in the hero (Three.js, with a photo fallback)
- Sections: hero, catalog, about, quality, contact (with map)

## Files
- `index.html` — the entire site (single static page, embedded CSS + JS)
- `images/` — placeholder product/factory photos (to be replaced with real Ursus photos)

## Notes
- Product names, descriptions and prices in `index.html` are sample data — replace with real catalog data.
- Contact phone/email are placeholders.
